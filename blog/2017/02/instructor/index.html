This is a instructor blog to teach you how to post a blog on this website.

  1. Install Hugo: install Hugo by this link.
  2. Clone the repo.
  3. Go to the repo directory.
  4. hugo new post/your-blog-name.md and generate a markdown file in directory /content/post
  5. Open the blog file and the generated config block is as following.
  6. date = “2017-02-13T09:08:31-08:00”
  7. title = “instructor”
  8. draft = true
  9. tags = [“tag1”,“tag2”]
  10. categories = [“category”]
  11. description = “”
  12. images = [“/2016/10/image.jpg”]
  13. Delete draft = true and change other configs like tags and category.
  14. Edit your blog and make sure the footer of your blog is the same as this instructor, which presents your name.
  15. Once you finish your blog, run hugo server -w and see the website on localhost:1313/blog/ to see if it is presented as you want.
  16. Once you think it works well, send pull request to commit.
